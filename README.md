# SmartTaskTracker
Multi-user task tracker built with Java, Spring Boot, Maven, and MySQL. Features JWT authentication via Spring Security, REST API documented with Swagger/OpenAPI, and object mapping with MapStruct. Designed for AWS deployment with RDS and Elastic Beanstalk.

1. Clone the repository
2. Copy application-example.yml to application.yml
3. Fill in your own database credentials
4. Run the application
